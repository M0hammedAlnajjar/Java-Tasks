# Java Programming Tasks (317–327)

## Overview
This repository contains solutions for **Java Programming Tasks 317–327**. These exercises focus on using loops, conditional statements, arithmetic operations, and basic algorithms in Java. Each program is implemented as a separate Java class following Java naming conventions and includes comments explaining the program logic.

## Learning Objectives
By completing these tasks, you will practice:

- Variables and data types
- Loops (`for` and `while`)
- Conditional statements (`if`, `else if`, `else`)
- Arithmetic operations
- Number manipulation
- Pattern-based algorithms
- Writing clean, readable Java code
- Proper commenting and indentation

---

## Tasks Included

| Task | Class Name | Description |
|------|------------|-------------|
| 317 | `FibonacciSequence` | Generates and displays the first 15 Fibonacci numbers. |
| 318 | `NumberReversal` | Reverses the digits of a given number. |
| 319 | `PalindromeChecker` | Checks whether a number is a palindrome. |
| 320 | `StudentGradeCalculator` | Calculates average grades and determines student status. |
| 321 | `DigitCounter` | Counts the number of digits in a number. |
| 322 | `LargestNumberFinder` | Finds the largest number from a set of values. |
| 323 | `EvenOddCounter` | Counts even and odd numbers within a given range. |
| 324 | `MultiplicationCalculator` | Calculates the multiplication of numbers within a specified range. |
| 325 | `DecimalToBinary` | Converts a decimal number to binary without using built-in methods. |
| 326 | `DigitFrequencyCounter` | Counts the frequency of a specific digit in a number. |
| 327 | `ArmstrongNumberChecker` | Determines whether a number is an Armstrong number. |

## How to Compile

Compile a Java file using:

```bash
javac ClassName.java
```

Example:

```bash
javac FibonacciSequence.java
```

---

## How to Run

Run the compiled class using:

```bash
java ClassName
```

Example:

```bash
java FibonacciSequence
```
